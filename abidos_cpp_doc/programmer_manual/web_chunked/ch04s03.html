4.3. Loader classes

Figure 4.3. c_loader UML diagram


The loader module stores the files that the parser will need, when the parser asks for the next file; loader module tries to search this file in the directories extracted from the .abidos_cpp/files_input. If not found that file in the fist try loader will try do it again in a case insensitive mode (due are the _lowercase classes)
